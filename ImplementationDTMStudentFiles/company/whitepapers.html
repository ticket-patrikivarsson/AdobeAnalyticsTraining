DTM DEBUGGER (DTM SWITCH)
Installation and Usage Instructions:

Firefox Plugin Install
1. Download the dtm_switch.xpi file from here: DTM Switch for FireFox
2. Drag the file into Firefox
Use
1. You should see the DTM icon on the top-right corner of your browser in the most recent version of FireFox. If not, find DTM Switch in Customize Menu (generally listed as the text "Hello") and place it on your toolbar. Once on your toolbar, it should appear as the Adobe Marketing Cloud icon.
2. Click on the icon
3. Toggle the buttons
4. You will need to refresh the page to see the changes


Chrome Plugin Install
1. Download the dtm_switch.crx file from here: DTM Switch for Chrome
2. Go to chrome://extensions in Chrome
3. Drag the file into the Chrome window
Use
1. Go to a web page that has DTM installed
2. You should see the DTM icon (the Adobe Marketing Cloud icon) on the top-right side of your address bar
3. Click on it
4. Toggle the buttons
5. You will need to refresh the page to see the changes
